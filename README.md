# C++ Game Engine
Creating a game engine from scratch to learn the fundamentals of game development. This is going to be a playground 
for me to learn low-level graphics programming and refine my coding style. The goal is to have a formal format for 
my internal system. 

## Getting Started
1. Clone the repository
2. Creat a build folder
3. Configure CMake
4. Build the project
5. Run the project
